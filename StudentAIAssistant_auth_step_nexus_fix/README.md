# AI-ассистент студента

Готовый стартовый Android-проект для Android Studio.

## Стек

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose

## Как открыть

1. Распакуй архив.
2. Открой Android Studio.
3. Нажми `File -> Open`.
4. Выбери папку `StudentAIAssistant`.
5. Дождись Gradle Sync.
6. Запусти приложение на эмуляторе или телефоне.

## Что уже реализовано

- Светлая тема в стиле проекта.
- Фиолетовый/индиго акцент.
- Нижняя навигация.
- Главный экран.
- Экран расписания.
- Экран конспектов.
- Экран AI-инструментов.
- Экран профиля.
- Общие UI-компоненты: карточки, дедлайны, пары, быстрые действия.
